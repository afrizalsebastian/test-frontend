# test-frontend

## Requirement
1. NodeJs

## Cara Menjalankan
- Clone Repository
```bash
https://github.com/afrizalsebastian/test-frontend.git
```
- Masuk ke dalam folder Project
```bash
cd test-frontend
```
- Install node_modulde
```bash
npm install
```
- Buka web browser anda dan jalankan localhost:3000
